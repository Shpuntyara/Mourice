# Операции над Морисом

Когда владелец просит изменить что-то в себе — делай именно так:

## Голос
**"Смени голос / поменяй голос / другой голос / голос из файла X"**
1. `read_file` → `C:/Users/sashk/Mourice/.env`
2. Заменить строку `MOURICE_SPEAKER_REFERENCE=...` на новый путь к .wav файлу
3. `write_file` → сохранить весь `.env` с изменённой строкой
4. Сообщить владельцу: нужно перезапустить бота чтобы изменение вступило в силу

Файлы голосов лежат в: `C:/Users/sashk/Mourice/voices/` и `C:/Users/sashk/Mourice/`

## Модель LLM
**"Смени модель / используй другую модель / переключись на X"**
1. `read_file` → `C:/Users/sashk/Mourice/.env`
2. Заменить строку `MOURICE_DEFAULT_MODEL=...` на название модели (например `qwen2.5:14b`)
3. `write_file` → сохранить весь `.env`
4. Сообщить: нужно перезапустить бота

## Параметры голоса (скорость, температура и т.д.)
**"Сделай голос быстрее / медленнее / выразительнее"**
1. `read_file` → `C:/Users/sashk/Mourice/.env`
2. Изменить нужный параметр (MOURICE_XTTS_SPEED, MOURICE_XTTS_TEMPERATURE и т.д.)
3. `write_file` → сохранить
4. Сообщить: нужно перезапустить бота

## Язык ответов
**"Отвечай по-английски / переключись на польский"**
Команда `/lang en` или `/lang pl` прямо в Telegram — перезапуск не нужен.

## Очистить историю диалога
**"Забудь всё / сбрось историю / начни заново"**
Команда `/reset` прямо в Telegram — перезапуск не нужен.
Или: `write_file` → `C:/Users/sashk/Mourice/data/conversation.json` с содержимым `[]`
